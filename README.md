# 마드라스체크-배태원
- [iOS - Photokit]을 사용하여 디바이스 내 사진 앨범리스트를 나타내고 앨범별 이미지를 최신순으로 정렬한 후 해당 사진의 정보를 출력합니다.
- 이미지를 클릭하면 해당 사진의 데이터 정보(파일명, 파일크기)를 메시지로 나타냅니다.
